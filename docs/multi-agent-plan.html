📌 核心结论

  • 推荐方案一（轻量级） - 使用 OpenClaw 子 Agent，10 分钟即可搭建完成

🚀 方案一：轻量级（推荐）

1创建 Agent 配置文件

# 创建 agents 目录 mkdir -p ~/.openclaw/workspace/agents # 创建工作助手配置 cat > ~/.openclaw/workspace/agents/work-agent.md << 'EOF' # 工作助手 (work-agent) ## 角色定位 - 专业领域：项目管理、会议安排、文档处理、邮件沟通 - 性格：严谨、高效、结果导向 - Emoji: 📊 ## 核心技能 - dingtalk_calendar - 日程管理 - dingtalk_docs - 文档处理 - summarize - 会议纪要 - github - 代码协作 ## 响应风格 - 简洁、专业、带行动项 - 优先使用表格和列表 - 重要事项主动提醒 EOF

2创建协调器脚本

# 创建 coordinator.py cat > ~/.openclaw/workspace/scripts/coordinator.py << 'EOF' #!/usr/bin/env python3 """多 Agent 协作协调器""" class AgentCoordinator: def spawn_agent(self, agent_name, task): # 启动子 agent cmd = ["openclaw", "sessions", "spawn", "--runtime", "subagent", "--label", agent_name, "--task", task] def send_to_agent(self, agent_label, message): # 发送消息到指定 agent cmd = ["openclaw", "sessions", "send", "--label", agent_label, "--message", message] EOF

3创建快捷命令

# 添加到 ~/.zshrc # 向工作助手发送任务 alias work='openclaw sessions send --label work-agent --message' # 向生活助手发送任务 alias life='openclaw sessions send --label life-agent --message' # 向技术专家发送任务 alias tech='openclaw sessions send --label tech-agent --message' # 查看 agent 状态 alias agents-status='openclaw subagents list --action list'

⚡ 方案二：进阶版（ACP Harness）

特性 方案一 方案二
复杂度 简单 中等
代码能力 基础 强大 (Codex/Claude)
启动时间 ~10 分钟 ~30 分钟
适用场景 日常协作 复杂项目

创建 ACP 会话

# 技术专家（使用 ACP，适合代码任务） openclaw sessions spawn \ --runtime acp \ --agentId codex \ --mode session \ --label tech-agent \ --task "资深技术专家，擅长代码审查、架构设计" \ --thread true

📊 任务管理表格

钉钉 AI 表格字段设计

字段名 类型 说明
任务 ID 文本 唯一标识，如 TASK-001
任务描述 文本 任务详细说明
负责 Agent 多选 work/life/tech-agent
状态 单选 待处理/进行中/已完成
优先级 单选 低/中/高/紧急
进度 文本 百分比或文字描述

💡 使用示例

单 Agent 任务

# 工作助手 - 安排会议 work "帮我安排下周的项目评审会议" # 生活助手 - 查询天气 life "周末上海天气怎么样？适合出游吗？" # 技术专家 - 代码审查 tech "帮我 review 一下这个 PR 的代码质量"

多 Agent 协作

# 场景：准备季度技术报告 # 1. tech-agent 收集技术指标 # 2. work-agent 整理报告格式 # 3. tech-agent 撰写技术内容 # 4. work-agent 最终审核发送

🔧 维护建议

  • 每周清理 - 检查并关闭不活跃的子 agent
  • 记忆整理 - 定期合并共享记忆中的重复信息
  • 权限审查 - 定期检查各 agent 的 API 权限
  • 性能监控 - 观察 token 使用情况，优化提示词

## ⚠️ 异常处理流程

### X 平台 Cookie 过期

**症状:**
- 无法访问 x.com/home
- 页面跳转到登录页
- 抓取到的互动数据全为 0

**处理:**
1. 报告中红色警告置顶
2. 推送提醒消息给用户
3. 降级使用 Google 搜索 + News Minimalist
4. 等待用户重新登录

**用户操作:**
```
1. 在 Chrome 中打开 x.com
2. 重新登录账号
3. 点击 Browser Relay 插件图标
4. 确认附加标签页
5. 通知助手恢复
```

## 3. 各平台关注点

### 3.1 X 平台

**筛选维度:**
- **排序方式:** top、latest、people、media、lists 综合
- **互动数据:** reply、repost、like、view 多维度加权
- **热度公式:** `score = like×1 + repost×2 + reply×3 + view×0.001`
- **时间范围:** 最近 3-7 天
- **语言:** 英文为主，重要中文内容纳入

**执行步骤:**
1. 搜索关键词（各领域的核心词）
2. 分别抓取 top 和 latest 排序结果
3. 提取完整互动数据
4. 计算热度分数
5. 按领域分类筛选
6. 人工复核内容质量

---

### 3.2 News Minimalist

**筛选标准:**
- **Significance 阈值:** >= 5.2（重要新闻）
- **来源验证:** 找到原媒体链接
- **阅读理解:** 抓取原文，提炼核心要点
- **去重:** 避免同一事件重复报道

**执行步骤:**
1. 访问 News Minimalist 各板块
2. 筛选 significance >= 5.2 的新闻
3. 提取原标题、摘要、significance 值
4. 追踪原媒体链接
5. 抓取原文深度阅读
6. 提炼核心要点 + 评价

---

### 3.3 YouTube

**重点栏目:**
- **播客 (Podcasts):** 深度访谈、行业对话
- **新闻 (News):** 科技新闻、时事分析

**筛选标准:**
- **频道质量:** 订阅数 > 10 万，专业度高
- **视频时长:** > 10 分钟（深度内容）
- **发布时间:** 最近 7 天
- **互动数据:** 点赞/观看比 > 3%

**推荐频道:**
- 播客：Lex Fridman, Joe Rogan, Naval Podcast, a16z
- 新闻：Bloomberg, CNBC, Reuters, TechCrunch

**执行步骤:**
1. 搜索关键词 + 筛选播客/新闻分类
2. 按上传时间和观看数排序
3. 提取标题、频道、时长、互动数据
4. 用 summarize 工具提取字幕/摘要
5. 提炼核心观点
